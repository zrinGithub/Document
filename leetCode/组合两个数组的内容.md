# 组合多个数组的内容

给定多个数组，返回所有可能的组合



请你找出这两个正序数组的中位数，并且要求算法的时间复杂度为 O(log(m + n))。

你可以假设 `nums1` 和 `nums2` 不会同时为空。

 

**示例 1:**

```
nums1 = ['男', '女']
nums2 = ['大号','小号']

返回组合：['男', '大号']['男', '小号']['女', '大号']['女', '小号']
```



**答案：**

和归并排序的思想是一样的，首先把两个数组合并到一起。

```java
class Solution {
     public static Set<String> combinationArrays(String[][] params) {
        Set<String> result = new LinkedHashSet<>();
        selectOne(params, 0, result, new String[params.length]);
        return result;
    }

    //指定数组里面获取一个元素
    public static void selectOne(String[][] params, int index, Set<String> result, String[] temp) {
        for (int i = 0; i < params[index].length; i++) {
            temp[index] = params[index][i];

            //已经是最后一个元素了，可以作为一条记录
            if (index == params.length - 1) {
                result.add(Arrays.toString(temp));
                continue;
            } else {
                //继续获取下一个元素
                selectOne(params, index + 1, result, temp);
            }
        }
    }

    public static void main(String[] args) {
        String[] gender = {"男", "女"};
        String[] size = {"XS", "S", "M", "L", "XL"};
        String[] color = {"红", "绿", "蓝", "黄", "棕"};
        Set<String> strings = combinationArrays(new String[][]{gender, size, color});
        System.out.println(strings);
    }
}
```
